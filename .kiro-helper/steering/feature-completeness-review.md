# Feature Completeness Review

For every feature, ask: Can a REAL USER do their REAL JOB?

## Land Acquisition — Acquisition Manager

Can they:
- [ ] Create an opportunity with full details?
- [ ] Edit an opportunity?
- [ ] View all opportunities in a list with search, filter, sort, pagination?
- [ ] View the pipeline board and drag cards between stages?
- [ ] View full opportunity detail with all tabs working?
- [ ] Transition status through the full lifecycle?
- [ ] Withdraw with a proper reason?
- [ ] Add/edit/delete a land owner?
- [ ] Create and manage due diligence checks?
- [ ] Submit offers and handle counter-offers?
- [ ] Upload and download documents?
- [ ] View financial feasibility?
- [ ] Request approval from Finance Director?
- [ ] Track the full activity/audit trail?
- [ ] See dashboard KPIs and charts?
- [ ] Receive notifications for key events?
- [ ] Export data as CSV?
- [ ] See their name and role correctly displayed?
